# A3StudentApp
For COMP 3005 Assignment 3, Part 1

This implementation is written in Java and uses JDBC for interacting with PostgreSQL. It was built using Maven.

Compiling the application:
`mvn compile`

Command to run the application:
`mvn exec:java -Dexec.mainClass=com.example.App`

*NOTE: Use cmd and not Windows Powershell*
